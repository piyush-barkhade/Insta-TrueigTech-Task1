InstaRight: An Instagram Clone (Task 1 TRUEiGTECH)


InstaRight is a fully functional, modern web application built to replicate core features of Instagram, including user authentication, real-time post feeds, like/unlike functionality, and comments (with timestamps). The application utilizes a robust MERN stack architecture.
